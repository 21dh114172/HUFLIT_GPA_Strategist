# Quy chế tính điểm GPA & Học vụ (HUFLIT)

Tài liệu này quy định các quy tắc tính toán điểm trung bình học kỳ, điểm trung bình tích lũy (GPA), xếp loại học lực, và các quy định về học lại/học cải thiện.

## 1. Thang điểm quy đổi (Thang 4.0)

| Điểm chữ | Thang 4.0 | Ghi chú |
| :------- | :-------- | :------ |
| **A+, A**| **4.0**   |         |
| **B+**   | **3.5**   |         |
| **B**    | **3.0**   |         |
| **C+**   | **2.5**   |         |
| **C**    | **2.0**   |         |
| **D+**   | **1.5**   |         |
| **D**    | **1.0**   |         |
| **F**    | **0.0**   | Rớt môn |

### Các trường hợp ngoại lệ (Không tính vào GPA)
- **Điểm P**: Đạt không phân mức.
- **Điểm I**: Hoãn thi.
- **Điểm X**: Chưa đủ dữ liệu.
- **Điểm R**: Miễn học được công nhận tín chỉ.
- Các học phần **không nằm trong yêu cầu** của chương trình đào tạo.

## 2. Công thức tính toán

Điểm trung bình học kỳ, năm học và tích lũy được tính theo công thức:

$$A = \frac{\sum (a_i \times n_i)}{\sum n_i}$$

- **A**: Điểm trung bình học kỳ/năm học/tích lũy.
- **$a_i$**: Điểm học phần thứ $i$ (quy đổi sang thang 4.0).
- **$n_i$**: Số tín chỉ của học phần thứ $i$.
- Kết quả được **làm tròn đến 2 chữ số thập phân**.

## 3. Quy định Học lại & Học cải thiện

Hệ thống chọn điểm chính thức để tính điểm tích lũy theo quy tắc:

### Học lại (Trường hợp điểm cũ là F)
- **Quy tắc**: Lấy điểm của **lần học cuối cùng**.
- **Mô tả**: Dù lần sau cao hay thấp hơn, điểm mới nhất sẽ thay thế điểm cũ trong tính toán GPA tích lũy.

### Học cải thiện (Trường hợp điểm cũ từ D đến C+)
- **Quy tắc**: Lấy điểm **cao nhất trong các lần học**.
- **Mô tả**: Sinh viên được cải thiện nếu điểm dưới B. Kết quả tốt nhất sẽ được giữ lại.

## 4. Xếp loại học lực & Tốt nghiệp

| GPA | Xếp loại |
| :-- | :------- |
| 3.60 - 4.00 | Xuất sắc |
| 3.20 - 3.59 | Giỏi |
| 2.50 - 3.19 | Khá |
| 2.00 - 2.49 | Trung bình |
| 1.00 - 1.99 | Yếu |
| < 1.00 | Kém |

### Hạ hạng bằng tốt nghiệp
Hạng tốt nghiệp (Xuất sắc, Giỏi) bị **giảm đi một mức** nếu:
1. Tổng số tín chỉ các môn học lại (**điểm F**) vượt quá **5%** tổng số tín chỉ toàn chương trình.
2. Bị kỷ luật từ mức cảnh cáo trở lên.

## 5. Hệ thống cảnh báo học tập

Dựa trên điểm trung bình học kỳ (để xét sau mỗi kỳ chính):

- **Học kỳ đầu tiên**: Cảnh báo nếu GPA học kỳ < **0.8**.
- **Các học kỳ tiếp theo**: Cảnh báo nếu GPA học kỳ < **1.0**.

Dựa trên GPA tích lũy:
- **Năm 1**: < 1.2
- **Năm 2**: < 1.4
- **Năm 3**: < 1.6
- **Năm tiếp theo**: < 1.8
